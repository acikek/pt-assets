## Español (es)

* En español, no hay un concepto de título del caso. Sólo se pone una mayúscula en la primera palabra.
  * Ej: "Block of Iron" -> "Bloque de hierro"
* Regional:
  * España:
    * lawrencio -> laurencio
    * telurio -> teluro
    * yodo -> iodo
    * circonio -> zirconio
    * wolframio -> volframio
  * Chile:
    * nihonio -> nihomio
  * Colombia:
    * tántalo -> tantalio
